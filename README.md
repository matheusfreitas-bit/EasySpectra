# EasySpectra

EasySpectra is a hyperspectral and multispectral processing toolkit with a graphical interface for spectral analysis, image alignment, preprocessing, and geospatial cube calibration.

## Main Features

- Hyperspectral cube import (TIFF, RAW, NPY)
- Band registration tools (manual + automatic)
- SuperGlue alignment integration
- RGB exploratory tools
- Spectral normalization
- GeoImport module (ODM integration)
- GUI built with Tkinter
- Compatible with macOS and Windows

## Running

Activate your virtual environment and run:

```
cd src
python -m easyspectra.interface
```
